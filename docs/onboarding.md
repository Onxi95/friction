# FocusGate Onboarding

1. Install the APK.
2. Open FocusGate.
3. Add domains and schedules.
4. Lock editing.
5. Start VPN and approve Android VPN permission.
6. In Brave, disable Secure DNS or configure it to use the current provider.
7. Optional: enable Always-on VPN in Android settings.

Always-on VPN path:

```text
Settings -> Network and Internet -> VPN -> FocusGate -> Always-on VPN
```

Do not enable `Block connections without VPN` for this DNS-only MVP.
